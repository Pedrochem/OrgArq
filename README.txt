Trabalho Final Organização e Arquitetura de Computadores I 2019/1; Professor Fabiano Hessel
Grupo Pedro Fratini Chem e Rafael Almeida de Bem

Este trabalho foi desenvolvido utilizando a linguagem Java. Como requisito, faz-se necessário que a JDK esteja instalada no computador do usuário.
Ferramentas utilizadas: Visual Studio Code, Git (GitHub), Vi e energético.

O processador deste trabalho é um processador MIPS de arquitetura Harvard multiciclo.

Para executar o simulador do processador MIPS, basta digitar no terminal (estando na pasta do trabalho):
$ ./runme.sh

O programa irá pedir um arquivo a ser simulado no formato .mips com as instruções em binário e os labels em plain text.

Exemplo de arquivo válido:

label:  00000001011010110101100000100001
        00000001011010110101100000100001
label2: 00000001011010110101100000100001
